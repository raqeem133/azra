Commitment to Accessibility

Azra is committed to ensuring that our website is accessible to all users, including those with disabilities.

Accessibility Features

  • Keyboard Navigation - Navigate the entire site using keyboard only
  • Screen Reader Support - Compatible with popular screen readers
  • Text Scaling - Adjust text size using browser settings
  • High Contrast Mode - Supports high contrast display settings
  • Skip Links - Jump directly to main content
  • WCAG 2.1 Compliance - Meets WCAG 2.1 Level AA standards

Accessibility Statement

We continuously work to improve the accessibility of our website and services. If you encounter any accessibility barriers, please contact us.
